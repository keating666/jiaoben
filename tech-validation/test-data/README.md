# 测试音频文件说明

## 所需音频文件

为了完成 MiniMax API 语音转文字功能验证，需要准备以下测试音频文件：

### 文件规格要求

1. **audio-30s.mp3** - 30秒测试音频
   - 格式: MP3
   - 时长: 30秒
   - 内容建议: 清晰的中文语音，如新闻播报或朗读
   - 文件大小: 约 500KB - 1MB

2. **audio-45s.mp3** - 45秒测试音频  
   - 格式: MP3
   - 时长: 45秒
   - 内容建议: 包含不同语速的中文对话
   - 文件大小: 约 750KB - 1.5MB

3. **audio-60s.mp3** - 60秒测试音频
   - 格式: MP3  
   - 时长: 60秒（系统支持的最大长度）
   - 内容建议: 完整的短视频脚本朗读
   - 文件大小: 约 1MB - 2MB

### 音频质量要求

- **采样率**: 16kHz 或 44.1kHz
- **比特率**: 128kbps 或更高
- **声道**: 单声道或立体声
- **背景噪音**: 尽量低
- **语言**: 中文普通话
- **语音清晰度**: 高

### 内容建议

#### 30秒音频内容示例
```
大家好，欢迎来到我们的短视频平台。今天我们要为大家介绍如何利用人工智能技术来创作优质的短视频内容。首先，让我们了解一下什么是AI语音识别技术。
```

#### 45秒音频内容示例  
```
人工智能正在改变我们的生活方式。从智能手机到智能家居，从自动驾驶到医疗诊断，AI技术无处不在。在短视频创作领域，AI也发挥着重要作用。它可以帮助我们进行语音识别、文本生成和内容优化。
```

#### 60秒音频内容示例
```
欢迎来到《科技前沿》栏目。今天我们要探讨的话题是人工智能在视频制作中的应用。随着技术的不断发展，AI已经能够实现语音转文字、自动字幕生成、智能剪辑等功能。这不仅提高了视频制作的效率，也降低了创作门槛。对于内容创作者来说，这是一个全新的机遇。让我们一起探索AI技术为视频行业带来的无限可能。
```

## 文件获取方式

### 方式1: 在线TTS服务生成
1. 使用百度、阿里云、腾讯云等TTS服务
2. 输入上述示例文本
3. 选择合适的声音和语速  
4. 下载对应时长的音频文件

### 方式2: 录音软件录制
1. 使用系统自带录音软件或Audacity
2. 朗读上述示例文本
3. 确保录音质量和时长符合要求
4. 导出为MP3格式

### 方式3: 免费音频资源
1. 访问freesound.org等免费音频网站
2. 搜索中文语音素材
3. 下载符合时长要求的文件
4. 根据需要进行剪辑处理

## 验证脚本

放置好音频文件后，可以运行以下命令进行验证：

```bash
# 验证文件是否存在
npm run test:minimax

# 或者单独测试每个文件
ls -la test-data/audio-*.mp3
```

## 注意事项

1. **版权问题**: 确保使用的音频内容没有版权问题
2. **文件路径**: 文件必须放在 `test-data/` 目录下
3. **文件命名**: 严格按照要求命名，区分大小写
4. **文件格式**: 目前只支持 MP3 格式
5. **网络环境**: 测试时确保网络连接稳定

## 测试流程

1. 准备好三个音频文件
2. 运行 `npm run test:minimax` 执行测试
3. 查看测试结果和性能指标
4. 检查日志输出中的错误信息
5. 根据结果调整API调用参数
