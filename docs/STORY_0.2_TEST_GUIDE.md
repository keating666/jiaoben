# Story 0.2 测试指南

## 🎯 测试目标
验证视频转文字编排服务的完整流程

## 📋 测试步骤

### 1. 准备测试数据

**抖音视频链接示例**：
- `https://v.douyin.com/iRyH8L8m/`
- 或任何有效的抖音分享链接

**API 密钥**（从你的 .env 文件获取）：
- MiniMax API Key
- 通义千问 API Key

### 2. 打开测试页面

访问：`https://jiaoben-2vch8wzkn.vercel.app/video-transcribe-dashboard.html`

### 3. 执行测试流程

1. **输入视频链接**
   - 在 "视频链接" 输入框中粘贴抖音链接

2. **输入 API Token**
   - 将你的 API 密钥组合输入
   - 格式：可以是 JSON 或逗号分隔

3. **选择脚本风格**
   - default（默认）
   - humorous（幽默）
   - professional（专业）

4. **设置语言**（可选）
   - 默认：zh（中文）

5. **点击 "开始处理"**

### 4. 预期结果

**成功流程**：
1. 显示 "正在处理..." 
2. 进度更新：
   - ✓ 提取视频链接
   - ✓ 下载视频
   - ✓ 提取音频
   - ✓ 语音转文字
   - ✓ 生成脚本
3. 显示最终脚本结果

**可能的错误**：
- "无效的视频链接" - 检查链接格式
- "API 认证失败" - 检查 API 密钥
- "处理超时" - 视频可能太长

### 5. 验证检查点

- [ ] 能成功提取抖音真实视频链接
- [ ] 能下载视频并提取音频
- [ ] MiniMax API 成功转写音频
- [ ] 通义千问 API 成功生成脚本
- [ ] 不同风格产生不同的脚本效果

## 🐛 故障排除

1. **页面 404**
   - 访问 Vercel Dashboard 获取最新 URL
   - 确认 public 文件夹中有 HTML 文件

2. **API 错误**
   - 检查浏览器控制台（F12）
   - 查看网络请求详情

3. **CORS 错误**
   - 这是正常的，API 应该返回正确的 CORS 头
