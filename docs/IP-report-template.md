# IP定位报告模板

## 第一部分：基本信息

**IP领域个人信息：** 性别年龄，职业经历，所在行业
**目标受众：** 详细描述目标受众
**受众痛点：** 详细描述目标受众的最痛的问题
**商业目标：** 受众的商业诉求

## 第二部分：商业IP的4个标准建议

**（内容固定）独特首创：** IP是独一无二的，生活处处是标准，为啥却更需要为生活

**（内容固定）创意核心一致：** 日更前，一定要核心理念是清晰的。

**（内容固定）好内容的标准：** 商业社群中，有受众加微信，且提出了针对业务的问题解答，就是好内容。

**（内容固定）价值观重于解答：** 当答不到精准时，跟同行业区别，你的看法态度比率就是解答方法

## 第三部分：定位及执行要点

**IP定位：** 一句话描述目标受众的定位

**（内容固定）发布频率：** 日更，日更，日更

**（内容固定）内容形式：** 原创为主（搭配账号技术及IP标准积累积累好）

**（内容固定）时长建议：** 30-40秒

## 第四部分：选题方向

IP所在**行业的竞争性的建议话题**

**行业分享性话题的议题

IP所在**行业最新的议题与趋势

IP所在**行业的受众服务标准/执行

IP所在行业与AI结合的建议议题

目测新季节营与IP所在行业结合的建议话题

IP的生活属内容：能反映出其隐藏个、话题营业等的品质

---

**注：带有"（内容固定）"标记的部分为固定内容，每个客户都使用相同内容。其他部分需要根据客户实际情况个性化填写。**
