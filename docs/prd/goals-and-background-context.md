# 1. 目标与背景上下文 (Goals and Background Context)

## 核心目标 (Goals)
* **验证核心价值**: 验证"AI仿写与生成" + "账号IP诊断"服务能有效解决内容创作者的效率和成功率痛点。
* **跑通商业闭环**: 验证通过"销售/客服引导 + 激活码"的模式能够成功转化付费用户。
* **搭建运营基础**: 建立一个功能完善的四角色管理后台，为后续的用户增长和运营打下基础。

## 背景上下文 (Background Context)
当前，短视频内容创作者普遍面临两大核心挑战：
1.  **方向定位问题**：许多创作者不清楚自己的内容应该往哪个方向发展才能匹配其IP定位并吸引目标受众。他们需要清晰、数据驱动的方向指引。
2.  **创作执行问题**：创作者希望模仿市场上的爆款视频，以走上一条成功确定性更高的路线，但往往缺乏高效、专业的仿写能力，导致耗时且效果不佳。

为应对这些挑战，本项目旨在打造一个AI驱动的SaaS平台。在第一期（MVP），我们将通过"**账号IP诊断报告**"为创作者指明内容方向，并通过核心的"**AI仿写与生成**"能力，帮助他们高效完成高质量的脚本创作。

在第二期，我们将进一步提升用户体验，新增"**行业热点追踪**"功能，并基于热点和用户的IP定位，每日为用户主动生成5条定制化脚本，实现从"工具"到"智能伙伴"的跨越。
