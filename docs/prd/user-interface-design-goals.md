# 3. 用户界面设计目标 (User Interface Design Goals)

## 整体UX愿景 (Overall UX Vision)
* 打造一个专业、高效且易于上手的创作工具。后台系统则应注重信息清晰度和操作便捷性，赋能销售和管理团队。

## 关键交互模式 (Key Interaction Paradigms)
* **C端用户**: 以"仪表盘"和"引导式表单"为核心。
* **B端后台 (销售/管理员)**: 以"数据表格"、"筛选器"和"看板"为核心。

## 核心界面和视图 (Core Screens and Views)
* **C端用户界面**: 主仪表盘, AI脚本生成页, 个人脚本管理列表, IP诊断报告查看页, 个人资料与账户设置。
* **销售人员后台**: 业绩与流程看板, 客户列表与筛选, 客户详情页, 为客户生成IP报告/仿写脚本的功能页, 激活码管理页。
* **管理员后台**: 全局数据看板, 用户与线索管理 (导入、分配), 兑换码管理 (生成、查看), 官网二维码配置。
* **超级管理员后台**: AI服务商API配置与管理, 管理员账户管理。

## 可访问性 (Accessibility)
* **标准**: 遵循 **WCAG AA** 标准。

## 品牌 (Branding)
* **假设**: 品牌风格暂未定义。初期建议采用简洁、专业、现代化的设计风格。

## 目标设备和平台 (Target Device and Platforms)
* **平台**: **Web Responsive** - 优先为桌面端浏览器设计，并良好地兼容主流平板和手机浏览器。
